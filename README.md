<h1>Virtual-City-Tour</h1>

<h2>Description</h2>
3D City Tour Game, an FPP 3D city tour game demonstrating a city tour with a player,  can be controlled by keys and mouse with the help of an integrated C# Script for the player's movement, built for both PC and Android.
<br />


<h2>Languages and Utilities Used</h2>

- <b>C#</b> 
<h2>Environments Used </h2>

- <b>Unity</b>
